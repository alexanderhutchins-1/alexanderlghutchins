ManchuOCR / Manchu_APP

ManchuOCR is an OCR and translation pipeline for the Manchu script. It integrates character segmentation, dictionary-based translation, and morphological parsing into an end-to-end workflow for digitizing historical texts.

Features

  • Segmentation of scanned Manchu pages
  • Romanization and dictionary translation
  • Interactive morphological analysis
  • Modular GUI with PySide6 and custom fonts
