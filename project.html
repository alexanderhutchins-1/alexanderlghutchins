ManchuOCR / Translation App

OCR · Romanization · Morphology · Translation

ManchuOCR is an end-to-end pipeline for the Manchu script: page segmentation, OCR, romanization, dictionary lookups, and morphology-aware translation.

Features

  • Segmentation of scanned Manchu pages
  • Romanization and dictionary translation
  • Interactive morphological analysis
  • Modular GUI (PySide6) with custom fonts
